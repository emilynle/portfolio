Projects

University Database

In this project, I created a website along with 2 other classmates that contained a database that holds professor records, student records, and other school-related entities with their respective attributes. We began by designing an ER Diagram and Relational Model (the schemas) to help the structure of our database entities along with their relations. Once completed, we utilized MySQL for the database tables. My classmate worked on the front-end design of the website using HTML5 and CSS3 to create the pages and forms to test our sample user queries.

Playdates

This project is a web-app that I worked on along with 3 other classmates where a user can create playdates between pets. Users can create playdates, accounts, sign in as well as sign out. They can view and join existing playdates created by other users.

Tuffy Types

Tuffy Types is a typing test website that I worked on along with 6 other classmates.The focus of the project was on the development process. We utilized the scrum and agile process known to be used in most software engineering projects. Through our scrum process, we used Trello's kanban boards to organize and manage our meetings and tasks. The result of this was a typing test, themed by CSU Fullerton's web design outline.
Not only did I help with code but I was also the scrum master of this team and learned how to manage a team of 7 as well as how to properly document a large scaled project.
